# Motion Detection System

This project implements a real-time motion detection system using Computer Vision techniques with Python and OpenCV. The system captures video from a webcam and detects moving objects by comparing consecutive frames. When motion is detected, the system highlights the moving region by drawing bounding boxes around it.

# Features
Real-time motion detection using webcam,
Frame differencing for detecting movement,
Noise reduction using Gaussian Blur,
Bounding boxes drawn around detected motion


# Technologies Used
Python,
OpenCV,
Computer Vision,
NumPy
